## Dezember 2020 (Version 6.34)

### &Uuml;berblick

Willkommen zur **Ver&ouml;ffentlichung vom Dezember 2020** :octicons-gift-16: von DietPi. Diese Version enth&auml;lt **5 neue Softwaretitel** :octicons-paper-airplane-16:, volle Unterst&uuml;tzung f&uuml;r den neu eingef&uuml;hrten **Raspberry Pi 400** und weitere interessante Funktionen und Fehlerbehebungen.

Es ist ein neues Softwarepaket, das in der optimierten DietPi-Liste enthalten ist. Portainer vereinfacht Ihr Docker-Container-Management &uuml;ber die Portainer-Weboberfl&auml;che. Es erm&ouml;glicht eine schnellere Bereitstellung der Anwendungen und bietet Transparenz in Echtzeit.

**[PaperMC - Hochoptimierter Minecraft-Server](../../software/gaming/#papermc)**

PaperMC erweitert die Liste der von DietPi unterst&uuml;tzten Minecraft-Server. Es ist in Java geschrieben und &uuml;ber Plugins erweiterbar, die jetzt zur Installation verf&uuml;gbar sind.

Weitere Einzelheiten finden Sie auf der [Dokumentationsseite](../../software/gaming/#papermc).

### &Auml;nderungen / Verbesserungen / Optimierungen

- **Aktivieren Sie automatische APT-Paketaktualisierungspr&uuml;fungen und aktualisieren Sie sie sogar t&auml;glich**. Durch Setzen der neuen `dietpi.txt`-Einstellung

`Sch
CONFIG_CHECK_APT_UPDATES=1
```

Sie k&ouml;nnen t&auml;gliche APT-Update-Pr&uuml;fungen aktivieren (Standardwert). Das Ergebnis wird im DietPi-Login-Banner angezeigt, &auml;hnlich wird ein verf&uuml;gbares DietPi-Update pr&auml;sentiert.

dietpi-automated_APT_package_update
![dietpi Automatisierte APT-Paketaktualisierung](../assets/images/dietpi-automated_APT_package_update.jpg){: width="670" height="326" loading="lazy"}

Satz

`Sch
CONFIG_CHECK_APT_UPDATES=2
```

um _APT-Pakete t&auml;glich automatisch_ aktualisieren zu lassen, wobei Protokolle in der n&auml;chsten Datei `/var/tmp/dietpi/logs/dietpi-update_apt.log` gespeichert werden. Cron wird f&uuml;r den Zeitplan verwendet, was bedeutet, dass Sie die Ausf&uuml;hrungszeit &uuml;ber `dietpi-cron > "cron.daily"` steuern k&ouml;nnen.

- **DietPi-Software** | **CloudPrint** :octicons-arrow-right-16: Diese Softwareoption wurde entfernt, da der Google Cloud Print-Dienst Ende 2020 eingestellt wird und wir keine Softwareoptionen anbieten m&ouml;chten, die f&uuml;r funktionieren h&ouml;chstens zwei Monate.
